# AjaiaDoc Product Overview

AjaiaDoc is a lightweight collaborative document editor built by Yahya Mundewadi for the Ajaia LLC Full Stack Product Engineer assessment.

## Key Features
- **Rich Text Formatting**: Headings, lists, bold, italic, quotes
- **File Upload**: Import .txt and .md files seamlessly
- **Sharing**: Granular Viewer and Editor access permissions
- **Version Snapshots**: Save & restore document history
- **PDF & Markdown Export**: One-click download

## Technology Stack
Built using Vanilla HTML5, CSS3, JavaScript, and Lucide icons.
