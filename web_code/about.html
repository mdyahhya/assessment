Project Specifications & Architecture

A comprehensive technical breakdown of AjaiaDoc — covering system flow, tech stack, database methods, and complete feature specifications.

1. Application Flow & Architecture
1
Authentication & Quick Test Sign-In
Users log in using password auth or select 1-click test accounts (user1@test.com as Owner or user2@test.com as Collaborator). Session state persists across refreshes.
2
Dashboard & Permission Filtering
Documents are dynamically rendered under two tabs: "My Documents" (owned) and "Shared with Me" (shared by email). Roles (Owner, Editor, Viewer) are computed and badge-encoded on card render.
3
Rich Text Canvas & Debounced Auto-Save
Opening a document loads the HTML content into an editable canvas. Input events trigger an active status badge ("Unsaved...") and debounce an auto-save operation after 1.5s of inactivity.
4
Version History Snapshotting
Clicking Save manually records an immutable snapshot with timestamp and title in the version history drawer. Any version can be restored in 1-click.
5
Multi-Format Document Exporting
Documents are exported client-side to styled A4 PDF documents via html2pdf.js or clean Markdown files (.md) via Blob downloads.
2. Tech Stack & Libraries Used
HTML5 Engine Vanilla CSS3 Design Tokens Vanilla JavaScript (ES6+) Lucide SVG Vector Icons html2pdf.js CDN jsPDF Library PostgreSQL Engine Supabase Backend API FileReader API LocalStorage Engine
3. Database Schema & RLS Policies
PostgreSQL Relations
Three primary tables: documents, document_shares, and document_versions with UUID keys and cascading deletes.
JWT-Based RLS Policies
Uses auth.jwt() ->> 'email' for shared document queries to prevent infinite policy recursion and guarantee instant response times.
Pre-Seeded Test Accounts
Seeds user1@test.com and user2@test.com directly into auth.users and auth.identities for instant reviewer testing.
4. Feature Specifications
✓
Rich Text Formatting
Headings H1-H3, Bold, Italic, Underline, Strikethrough, Bullet & Numbered lists, Quotes, and Clear Formatting.
✓
File Import (.txt and .md)
Drag-and-drop or browse files. Automatically parses markdown headers and text paragraphs into rich text DOM nodes.
✓
Role-Based Sharing
Grant collaborator permissions by email with Viewer (read-only) or Editor (can edit) access controls.
✓
Version History Drawers
Creates timestamped version snapshots on manual save with 1-click instant content restoration.
✓
PDF & Markdown Export
Generate styled A4 PDF documents or download clean .md files directly from the topbar.
✓
Responsive Light Theme
Minimalist white and slate design system with dynamic glassmorphism and Lucide SVG vector iconography.
