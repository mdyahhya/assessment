# Product Overview — AjaiaDoc

AjaiaDoc is a lightweight collaborative document editor inspired by Google Docs.

## Key Capabilities
- Rich-text formatting (Bold, Italic, Headings, Lists)
- File Upload & Conversion (.txt and .md)
- Access Sharing (Viewer vs Editor roles)
- Persistence via Supabase Postgres
- Version History Snapshots

## Stack
Built with React, Vite, Tiptap, Supabase, and Vercel by Yahya Mundewadi (yahya.in).
